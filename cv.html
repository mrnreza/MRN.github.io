Projects

  • Improving the performance of upside-down reinforcement learning: Upside-down reinforcement learning (UDRL) solves RL problems using supervised learning methods. Several modifications were proposed to improve exploration and sample efficiency including entropy bonus, using noisy layers, intrinsic reward according to the error of the dynamic model, using recurrent neural networks to recognize sequential patterns, and proportional number of updates.

  • Implementing artificial curiosity in deep reinforcement learning: Artificial curiosity (AC) encourages agents to search for novel areas and improves performance. Different methods of AC including visitation count and prediction error were implemented in different RL algorithms such as DQN, DDPG, D4PG, PPO, and SAC.

  • Deep Clockwork RNN: In Clockwork RNN, the hidden layer is divided into modules with different clock periods. Therefore, the network is able to model different patterns in sequential data. We proposed Deep Clockwork RNN which the network can consist of several hidden layers each with a different number of modules, time periods, and the number of neurons.

  • Implementing sliding mode controller on 6R robot with EtherCAT protocol: We used arduino with EasyCat Board as slaves and computer as a master in EtherCAT protocol to control the 6R robot using SMC method. Repeatability improved due to the faster and safer communication compared to UART.

  • Simulation and control of 6R robot with Leap Motion in ROS: Leap motion sensor can detect hand position and motion. we use it to control the 6R robot with hand in simulation and experiment in ROS.

  • Automatic path planning of spraying robot: We used RoboDK software to model the robot and spraying process and export robot program. An algorithm was developed to plan an acceptable path for spraying different parts completely. The user can use basic commands to generate and modify a path.

  • Dynamic modeling and load-carrying capacity of Hyundai industrial manipulator: Kinematic and dynamic models were obtained. According to the model and motors, dynamic load-carrying capacity was estimated and compared to the nominal payload. Our estimation is 163.5 kg and the nominal payload is 165 kg.

  • Kinematic, dynamic, and control analysis of 3UPS/S robot: Kane method was used in deriving the dynamic model of this parallel mechanism. Kinematic and dynamic models were evaluated with ADAMS software. PID and computed torque methods were simulated for controlling the robot.

  • Controlling 2d drawing CNC: As a team, we designed and fabricated a drawing CNC capable of multi colors. I was in charge of electronics and programming. I used a modified version of MARLIN and GRBL frameworks.
